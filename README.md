# -ALMA-LMS-Learning-Management-System
**ALMA LMS** is a modern, database-driven Learning Management System designed to simplify and automate online learning.   It enables **students** to register, enroll in courses, and make payments seamlessly, while **administrators** can manage users, courses, and transactions through an intuitive dashboard.
